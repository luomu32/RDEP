[[overview-http-status-codes]]
== HTTP状态码

|===
| Status code | Usage

| `200 OK`
| 请求成功，处理成功

| `400 Bad Request`
| 请求参数错误。如缺少必要的参数；参数类型错误；参数校验错误等

| `403 Forbidden`
| 访问受限的资源，需要通过身份认证

| `404 Not Found`
| 请求资源不存在

| `415 Unsupported Media Type`
| 不支持的媒体类型

| `405 Method Not Allowed`
| 不支持的Http Method。如仅支持POST，而请求却是GET

| `500 Internal Server Error`
| 服务错误

| `503 Service Unavailable`
| 服务暂不可用
|===
